1 / 1

JavaScript에는 중복을 제거하는 다양한 방법이 있습니다. 아래에 몇 가지 예시를 제시해 드리겠습니다.

  1. Set을 사용하여 중복 제거하기:
javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = [...new Set(array)];
console.log(uniqueArray); // [1, 2, 3, 4, 5]
  1. filter() 메서드를 사용하여 중복 제거하기:
javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = array.filter((value, index, self) => {
  return self.indexOf(value) === index;
});
console.log(uniqueArray); // [1, 2, 3, 4, 5]
  1. reduce() 메서드를 사용하여 중복 제거하기:
javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = array.reduce((accumulator, value) => {
  if (!accumulator.includes(value)) {
    accumulator.push(value);
  }
  return accumulator;
}, []);
console.log(uniqueArray); // [1, 2, 3, 4, 5]

이러한 방법을 사용하여 JavaScript에서 배열에서 중복을 제거할 수 있습니다. 각각의 방법은 조금 다른 접근 방식을 가지고 있으므로 사용하고자 하는 상황에 맞게 선택할 수 있습니다.

javascript

    backgroundAttachment: 'fixed',
    backgroundPosition: 'center',

    backgroundSize: 'cover',
  };
}

위의 코드에서 review-container 클래스에는 :class 디렉티브를 사용하여 동적으로 클래스를 할당하였습니다. ['review-container', { 'd-none': !movieDetail }, { 'd-lg-block': movieDetail }] 부분은 배열과 객체를 사용하여 클래스를 결정합니다.

  • review-container: 항상 적용되는 클래스입니다.
  • 'd-none': !movieDetail: movieDetail이 없는 경우(movieDetail이 falsy한 경우)에는 d-none 클래스가 적용되어 요소를 숨깁니다.

d-none 및 d-lg-block 클래스를 동적으로 계산하려면 computed 속성을 사용할 수 있습니다. 아래의 코드 예제를 참고해주세요:

html
<div :class="['reive-movie-inform', is_movieDetail ? 'd-lg-block' : 'd-none']">
  <MovieItem :item="movieDetail" />
</div>

위의 코드에서 :class 디렉티브를 사용하여 동적으로 클래스를 할당합니다. ['reive-movie-inform', is_movieDetail ? 'd-lg-block' : 'd-none'] 부분에서 배열과 삼항 조건 연산자를 사용하여 클래스를 결정합니다.

  • 'reive-movie-inform': 항상 적용되는 클래스입니다.
  • is_movieDetail ? 'd-lg-block' : 'd-none': is_movieDetail이 참인 경우(is_movieDetail이 true인 경우)에는 d-lg-block 클래스가 적용되고, 그렇지 않은 경우(is_movieDetail이 false인 경우)에는 d-none 클래스가 적용됩니다.

이렇게 하면 is_movieDetail 값에 따라 reive-movie-inform 클래스와 d-none, d-lg-block 클래스가 동적으로 조합되어 요소에 할당됩니다. is_movieDetail이 참이면 reive-movie-inform 요소가 화면에 표시되고 d-lg-block 클래스가 적용되며, 그렇지 않은 경우에는 d-none 클래스가 적용되어 요소가 숨겨집니다.
